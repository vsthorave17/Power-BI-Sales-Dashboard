# Power-BI-Sales-Dashboard
This Power BI project delivers a comprehensive Sales & Profit Performance Dashboard designed to help stakeholders monitor business performance across products, categories, payment methods, time periods, and sales representatives.
The dashboard highlights key KPIs such as Total Sales (4.14M), Total Profit (3.54M), Total Orders (3002), Average Order Value (1.38K), and Total Discounts Given (76.67K). It provides interactive insights into monthly sales vs. profit trends, category-wise performance, top and bottom products, sales representative contribution, and payment method distribution.
By combining high-level metrics with granular product and category analysis, the report supports data-driven decision-making, profit optimization, and performance tracking.

🛠 Tech Stack
SQL Server (SSMS): Data ingestion, cleaning, modeling, and analysis
Power BI Desktop: Used for data modeling, DAX calculations, and building interactive dashboards.
DAX (Data Analysis Expressions): Created measures and KPIs such as Total Sales, Total Profit, Average Order Value, Discounts, and comparative analytics.
Data Visualization: Used charts, tables, KPI cards, slicers, and filters for intuitive business insights.
Export & Documentation: Dashboard exported as .pbit for sharing and documentation purposes.

Data Source:
Kaggle Dataset.

✨Features and Highlights:
Interactive KPI Dashboard
Sales vs Profit Trend Analysis
Category‑Wise Performance Insights
Top & Bottom Product Analysis
Sales Representative Performance Tracking
Payment Method Distribution
Dynamic Filtering & Slicers
